# Socket

## Socket基本概念

1.什么是Socket

- 定义：Socket是一个通信端点，允许在同一台主机或通过网络连接的不同主机上的应用程序之间进行通信
- 本质：Socket是一种IPC机制，是网络编程的统一接口

2.通信域

通信域决定了：

- 通信范围：本机通信还是跨网络通信
- 地址格式：如何标识Socket

3.Socket类型

- 流Socket（TCP）
- 数据报Socket （UDP）

## 流Socket

1.核心特点

- 可靠传输：保证数据按顺序、无重复、无丢失地到达
- 双向通信：连接建立后，两端可以发送和接收数据
- 字节流：无消息边界，类似管道
- 面向连接：通信前需要建立连接

2.系统调用

- 创建 Socket，返回文件描述符:`int socket(int domain, int type, int protocol);`
- 将 Socket 绑定到本地地址:`int bind(int sockfd, const struct sockaddr *addr, socklen_t addrlen);`
- 将 Socket 变为监听状态，接受连接请求:`int listen(int sockfd, int backlog);`
- 接受客户端连接，返回新的Socket用于通信:`int accept(int sockfd, struct sockaddr *addr, socklen_t *addrlen);`

## 数据报Socket

1.核心特点

- 不可靠传输：可能丢失、重复、乱序
- 五连接：发送数据前无需建立连接
- 面向消息：保留消息边界，一次`sendto()`对应一次`recvfrom()`
- 每个报文独立路由

2.系统调用

- 向指定地址发送数据报：`ssize_t sendto(int sockfd, const void *buf, size_t len, int flags, const struct sockaddr *dest_addr, socklen_t addrlen);`
- 接收数据报，返回发送端地址:`ssize_t recvfrom(int sockfd, void *buf, size_t len, int flags, struct sockaddr *src_addr, socklen_t *addrlen);`

3.`connect()`

- 发送：可使用`write()`代替`sendto()`，目标地址为固定已连接的地址
- 接收：`read()`只接受来自对等地址的数据报
- 异步错误：连接后的数据报Socket可以接收异步错误
